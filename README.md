# DanmaBoy - Danmaku GameBoy

在直播间里用弹幕来玩GBA游戏。
demo地址：[douyu.com/lisp][douyu-url]


## 主要功能

 - [x] [申请直播间][douyu-url]
 - [x] [获取直播间弹幕][danmu]
 - [x] [程序调用GBA模拟器][keyboard]
 - [x] 处理弹幕逻辑，转化为输出信号


[douyu-url]: http://www.douyu.com/lisp
[danmu]: https://github.com/littlecodersh/danmu
[keyboard]: https://github.com/boppreh/keyboard
